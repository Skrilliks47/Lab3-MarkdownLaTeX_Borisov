# Таблицы в Markdown

## Команды Git

| Функция | Команда Git | Описание |
|:--------|:-----------:|---------:|
| Создание репозитория | `git init` | Инициализация нового репозитория |
| Просмотр состояния | `git status` | Показывает изменённые файлы |
| Добавление файлов | `git add .` | Добавляет все файлы в staging area |
| Создание коммита | `git commit -m "message"` | Сохраняет изменения |
| Отправка на GitHub | `git push` | Загружает коммиты на сервер |
| Получение изменений | `git pull` | Скачивает изменения с сервера |

## Элементы Markdown

| Элемент | Синтаксис | Пример |
|:--------|:---------:|--------|
| Заголовок | `# H1` | # Заголовок |
| Жирный | `**text**` | **жирный** |
| Курсив | `*text*` | *курсив* |
| Список | `- item` | - пункт |
| Ссылка | `[text](url)` | [GitHub](https://github.com) |
