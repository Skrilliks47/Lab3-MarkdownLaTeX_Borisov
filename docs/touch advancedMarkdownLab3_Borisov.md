## Alert-блоки GitHub

> [!NOTE]
> Это простая заметка о возможностях Markdown.

> [!TIP]
> Для экранирования специальных символов в Markdown используйте обратный слэш `\`.

> [!IMPORTANT]
> При работе с GitHub убедитесь, что файл `.gitignore` настроен корректно.

> [!WARNING]
> Не забывайте делать коммиты после каждого значимого изменения.

> [!CAUTION]
> Будьте осторожны при использовании `git push --force` — это может перезаписать историю.

---

## Escape-символы (экранирование)

- Заголовок без форматирования: \# Это не заголовок
- Звёздочка как текст: \*просто звёздочка\*
- Подчёркивание как текст: \_просто подчёркивание\_
