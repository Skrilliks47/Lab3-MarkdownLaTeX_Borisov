# Блоки кода и цитаты

## Inline-код

Команда `git status` показывает состояние репозитория, а `git add .` добавляет все изменённые файлы.

## Блок кода с указанием языка (C#)

```csharp
using System;

class Program
{
    static void Main()
    {
        Console.WriteLine("Hello, World!");
    }
}
```
## Блок кода без указания языка
git init

git add .

git commit -m "Initial commit"

git push
