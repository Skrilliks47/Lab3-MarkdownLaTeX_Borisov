# Форматирование текста в Markdown

## Основные типы форматирования

| Формат | Синтаксис | Результат |
|--------|-----------|-----------|
| **Жирный** | `**текст**` или `__текст__` | **жирный текст** |
| *Курсив* | `*текст*` или `_текст_` | *курсив* |
| ***Жирный курсив*** | `***текст***` | ***жирный курсив*** |
| ~~Зачёркнутый~~ | `~~текст~~` | ~~зачёркнутый~~ |
| `Код` | `` `код` `` | `код` |

---

## Комментированная программа на C#

Создайте консольное приложение на C#, которое демонстрирует все форматы Markdown в комментариях к коду.

### Требования к программе:
1. Имя файла: `FormatDemo.cs`
2. Логика:
   - Запрашивает у пользователя два числа
   - Выполняет их сложение
   - Выводит результаты в форматированном виде

### Пример структуры кода с комментариями в стиле Markdown:

```csharp
/**
 * ## Программа сложения двух чисел
 * 
 * **Описание:** Программа запрашивает у пользователя два числа
 * и выводит их сумму.
 * 
 * ### Алгоритм:
 * 1. Запросить первое число
 * 2. Запросить второе число
 * 3. Вычислить сумму
 * 4. Вывести результат
 * 
 * ~~~ Исправлено: обработка ошибок ~~~
 */

using System;

class FormatDemo
{
    static void Main()
    {
        // **Ввод первого числа**
        Console.Write("Введите первое число: ");
        double number1 = Convert.ToDouble(Console.ReadLine());
        
        // *Ввод второго числа*
        Console.Write("Введите второе число: ");
        double number2 = Convert.ToDouble(Console.ReadLine());
        
        // `Суммирование`
        double sum = number1 + number2;
        
        // ***Вывод результата***
        Console.WriteLine($"Результат сложения: {sum}");
    }
}
